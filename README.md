#### Springboard Data Analytics Track, Capstone Project One 
##### 2012 Global PISA Findings and Data Visualization
This Capstone One project used a univariate, data analysis (including Google's Colab, i.e., essentially Jupyter Notebooks) for the data visualization(s) of global 2012, OECD PISA rankings and scores. HTML was used for displaying the Exploration Slideshow's findings via Matplotlib; Numpy; Pandas; Python and Seaborn (for presenting the graphical visualizations herein to executive, technical and not-technical audiences).
